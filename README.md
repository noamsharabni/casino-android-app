# Casino Android App

Full-stack multi-game casino platform built in Java for Android.

## Games
- Blackjack
- Poker (offline + real-time online multiplayer)
- Roulette
- Slot Machine
- Tic-Tac-Toe

## Technologies
Java, Android Studio, Firebase Realtime Database, SQLite, Gemini AI API, XML Layouts, OOP

## Key Features
- Dual-database architecture: Firebase (cloud) + SQLite (local) with full two-way sync
- Real-time online multiplayer Poker rooms (Create Room / Join Room)
- Gemini AI strategy assistant for Blackjack and Poker
- Full user authentication: Register, Login, session management
- Multi-threaded dealer animations using Thread and Handler
- Polished UI with custom animations (card flip, fade, zoom) and splash screen
- Real-time leaderboard (Casino King)

## Architecture
21 Java classes with clean OOP design — deployed and tested on a real Android device
